## アプリ概要
- **アプリ名**: タスクリー / Taskly
- **用途**: グループタスク管理アプリ（App Store 一般販売向け）
- **技術**: React Native + Expo + Supabase
- **課金モデル**: フリーミアム → 買い切り

## コア機能（開発予定）
- ✅ タスク管理（マイタスク・チーム全体）
- ✅ ルーティン管理（毎日/毎週/毎月）
- ✅ アカウント・チーム管理
- 🔲 チーム作成・招待フロー
- 🔲 課金（RevenueCat / Stripe）
- 🔲 App Store 申請
